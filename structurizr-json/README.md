# Structurizr JSON

This directory contains the definition of the JSON schema for describing software architecture models based
upon the [C4 model](https://c4model.com), and is the data storage format used by the [Structurizr tooling](https://docs.structurizr.com).

See [docs.structurizr.com/openapi](https://docs.structurizr.com/openapi) for details.
